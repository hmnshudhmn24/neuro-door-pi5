## ðŸš€ Quick Start (4 Steps)

1. **Extract**: `tar -xzf neurodoor-pi5.tar.gz && cd neurodoor-pi5`
2. **Install**: `sudo ./setup.sh install`
3. **Enroll Admin**: `python3 enroll_user.py`
4. **Start**: `sudo systemctl start neurodoor`
